# Mobile Version

## Overview
This directory contains the compiled version of the Kirebos Networks application for mobile platforms.

## Getting Started
To install and run the mobile version:
1. Transfer the APK file to your Android device, or the IPA file to your iOS device.
2. Follow the installation prompts on your device.
