# Kirebos Networks - Maintenance Sheet

## Preventive Maintenance
- **Backups**: Run the `backup.sh` script weekly to back up configurations.
- **Log Monitoring**: Regularly check `logs/kirebos_networks.log`.

## Corrective Maintenance
- **Restoration**: Use `restore.sh` to restore configurations in case of a failure.
- **Updates**: Run `update.sh` to get the latest versions.
